¿Qué significa HTTPS y qué es un certificado SSL?

HTTPs significa Hiper Text Transfer Protocol Secure y es una versión actualizada y más segura que HTTP. La S significa que el sitio web tiene un certificado SSL para crear una conexión segura. El protocolo HTTPS cifra los datos entre el sitio web (servidor) y el navegador (cliente). Los certificados SSL agregan una capa adicional de seguridad para los usuarios de su sitio web y aumentan la credibilidad.
